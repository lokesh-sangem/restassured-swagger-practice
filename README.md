# Altoro Mutual API Test Framework

## Setup Instructions
1. Clone the repository
2. Configure environment properties
3. Run tests using Maven or scripts

## Test Execution
```bash
mvn clean test
./scripts/run-tests.sh
-------------------------------------------------


1. **Models** - Data transfer objects (DTOs)
2. **Config** - Environment and test configuration
3. **Utils** - Reusable helper methods
4. **Constants** - Static values and endpoints
5. **Base** - Common test setup and teardown
6. **Tests** - Actual test implementations
7. **Listeners** - Test execution hooks
8. **Reporters** - Custom reporting
9. **Runners** - Test execution entry points
